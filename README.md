# Classification

This folder contains a summary of **Chapter 3: Classification** from *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow* (2nd Edition) by Aurélien Géron.

## Contents

| File | Description |
|---|---|
| `plan.md` | Full summary of Chapter 3 — covers MNIST, binary/multiclass/multilabel/multioutput classification, performance measures (confusion matrix, precision, recall, F1, ROC), precision/recall tradeoff, error analysis |
| `image-materials/` | Supporting diagrams (decision threshold, ROC curves, etc.) |
| `github-update.md` | Quick reference for pushing changes to GitHub via terminal |

## Topics Covered

- MNIST dataset
- Binary classification (SGDClassifier)
- Performance metrics: accuracy, confusion matrix, precision, recall, F1 score
- Precision/Recall tradeoff & decision threshold
- ROC curve & AUC
- Multiclass classification (OvA, OvO)
- Error analysis
- Multilabel & multioutput classification
